# Лазерный Сейф — Взлом трёхзначного кода (Pygame)
Проект моделирует работу механического кодового сейфа.  
Пользователь вводит трёхзначный код, диск сейфа плавно вращается к выбранной цифре,  
а программа проверяет введённую комбинацию и визуально показывает результат.

Проект выполнен на **Pygame**, включает интерфейс, анимацию и логику взлома.

---

## 📦 Возможности

### 1. Ввод трёхзначного кода  
На экране расположены три кликабельные цифровые кнопки (0–9).  
Каждый клик увеличивает значение соответствующей цифры.  
Выбранные значения хранятся внутри программы и используются для проверки комбинации.

### 2. Сейфовый диск  
В правой части окна отображается круг — кодовый диск сейфа.  
По окружности расположены десять делений с цифрами от 0 до 9.  
Все координаты вычисляются через тригонометрию, чтобы элементы были равномерно распределены.

### 3. Плавная анимация вращения  
При изменении первой цифры диск начинает плавно вращаться к соответствующему углу.  
Анимация реализована через постепенное изменение текущего угла и перерисовку кадра.  
Поворот выглядит как движение настоящего механического сейфа.

### 4. Проверка кода и визуальная индикация  
При запуске программа генерирует случайный трёхзначный код.  
После нажатия кнопки «Проверить код» введённая комбинация сравнивается с секретной.  
Если код совпал — диск подсвечивается зелёным.  
Если неверный — диск становится красным.

## 🚀 Установка зависимостей

Установите Pygame:

```bash
pip install pygame
source .venv/bin/activate
pip install -r requirements.txt
```
## Запуск
```bash
python LaserSafe.py
```
## Использование
1. Запустите программу.
2. Нажимайте на три цифровые кнопки слева, чтобы выбрать код.
3. Диск автоматически вращается к выбранной первой цифре.
4. Нажмите кнопку «Проверить код».
5. Если код совпал — диск станет зелёным.
6. Если нет — диск станет красным.
